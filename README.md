# JS-Fundamentals

This repository contains beginner-level JavaScript exercises for learning purposes.

## Exercise 0: First constant, first print

Prints "JavaScript is amazing" to the console.

### Requirements

- Create a constant variable called `myVar` with the value `"JavaScript is amazing"`.
- Use `console.log(...)` to print the value.
- Do **not** use `var`.

### How to Run

1. Make sure [Node.js](https://nodejs.org/) is installed.
2. Open terminal or Git Bash.
3. Navigate to this folder.
4. Run: node 0-javascript_is_amazing.js


You should see: JavaScript is amazing
